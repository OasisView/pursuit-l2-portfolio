## Projects

### 1. Apple TV Prototype
**Role:** Frontend Developer

A streaming platform prototype powered by 4 machine learning recommendation models. I built the entire React frontend — movie browsing, search, personalized recommendation display, login/signup flows, and full responsive styling — while integrating with a Flask backend and ML API.

**Tech:** React, Vite, Tailwind CSS, React Router

### 2. Slack Clone MVP
**Role:** Frontend Developer

A real-time messaging platform with channels, user auth, and persistent message history. I built all frontend UI components, pages, and styling using React and CSS Modules, connected to a WebSocket-powered Express backend.

**Tech:** React, CSS Modules, Socket.io (client), JWT Auth
**Teammate:** [Ismael Caraballo](https://github.com/ismaelcaraballo-afk) (Backend & WebSockets)

[View Repository →](https://github.com/OasisView/slack-clone)

### 3. BabbleClone (LingoVision)
**Role:** Backend Developer

A visual-first Spanish language learning app that teaches through images and audio — no translations. I built the Express REST API, set up the Supabase PostgreSQL database with Row Level Security, implemented the CI/CD pipeline with GitHub Actions, and wrote end-to-end tests with Playwright.

**Tech:** Node.js, Express, Supabase (PostgreSQL), GitHub Actions, Playwright
**Teammate:** [Michael](https://github.com/newbloomwon) (Frontend & UI)

[View Repository →](https://github.com/OasisView/BabbleClone)

---

### 4. RateShield NYC
**Role:** Frontend Developer

A Con Edison bill transparency tool for NYC residents. Enter three numbers from your bill and get a plain-English breakdown of what you control vs. what's driven by the grid and regulators. Includes energy-saving tips, links to active legislation, and assistance programs.

**Tech:** Next.js 14, TypeScript, Tailwind CSS, Recharts, Zustand, Supabase, React Hook Form
**Teammate:** [Joshua Viera](https://github.com/JoshuaViera) (Backend)

[View Repository →](https://github.com/OasisView/rateshield) | [Live Site →](https://rateshield.vercel.app)
